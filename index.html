English documentation (Of course!)
Option name Default value Description
address_suffix "" (empty string) Suffix to add at the end of the address before sending it to the geocoder.
Example: ", France" if you want to search after french addresses.
bind_marker true Bind a marker on the map.
draggable_marker true Let the user move the marker on the map.
region_bias null To influence the result sent by the geocoder in accordance with a region.
For example, if region_bias is "fr", a search on "Paris" keyword will send you back the french capital instead of the city in Texas or Tennessee.
update_callback null Not used for the moment.
map_options.min_zoom 4 Minimum zoom level
map_options.max_zoom 18 Maximum zoom level.
map_options.tile.url "http://{s}.tile.cloudmade.com/BC9A493B41014CAABB98F0471D759707/997/256/{z}/{x}/{y}.png"
map_options.tile.attrib "Map data © openstreetmap contributors"
map_options.zoom 5 Initial zoom level.
map_options.center new L.LatLng(46, 2) Default centring.
map_options.scroll_wheel false Authorize the user to zoom by scrolling.
elements.map false Tag within which the map will be.
elements.lat false Tag within which the latitude will be displayed.
elements.lng false Tag within which the longitude will be displayed.
elements.locality false Tag within which the city will be displayed.
elements.administrative_area_level_1 false Tag within which the administrative area level 1 will be displayed.
For example States in USA, "Région" in France.
elements.administrative_area_level_2 false Tag within which the administrative area level 2 will be displayed.
Departments in France.
elements.country false Tag within which the country will be displayed.
elements.zip_code false Tag within which the zip code will be displayed.
elements.type false Tag within which the view type will be displayed. For example: locality, establishment, street address, etc.
About

Le projet jQuery Leaflet addresspicker est inspiré de jQuery addresspicker qui fait globalement la même chose, mais pour google maps.

Je travaillais sur un projet leaflet et voulais avoir quelque chose comme jQuery addresspicker, donc j'ai eu l'idée de faire leaflet addresspicker.

Si vous voulez contribuer, faites un fork du projet et envoyez une pull request :).



The project jQuery Leaflet addresspicker was inspired by jQuery addresspicker which is nearly the same plugin for google maps.

I was working on leaflet and wanted to have something like jQuery addresspicker, so i did leaflet addresspicker.

If you want to contribute, please fork the project and do a pull request :).
