Documentation (english)

Option name Default value Description
address_suffix "" (empty string) Suffix to add at the end of the address before sending it to the geocoder.
Example: ", France" if you want to search after french addresses.
bind_marker true Bind a marker on the map.
draggable_marker true Let the user move the marker on the map.
region_bias null To influence the result sent by the geocoder in accordance with a region.
For example, if region_bias is "fr", a search on "Paris" keyword will send you back the french capital instead of the city in Texas or Tennessee.
map_options Object Some options about the map and its initialisation.
map_options.min_zoom 4 Minimum zoom level
map_options.max_zoom 18 Maximum zoom level.
map_options.tile.url "http://{s}.tile.cloudmade.com/BC9A493B41014CAABB98F0471D759707/997/256/{z}/{x}/{y}.png"
map_options.tile.attrib "Map data © openstreetmap contributors"
map_options.zoom 5 Initial zoom level.
map_options.center new L.LatLng(46, 2) Default centring.
map_options.scroll_wheel false Authorize the user to zoom by scrolling.
elements Object Some HTML tags containing datas to update.
elements.map false Tag within which the map will be.
elements.lat false Tag within which the latitude will be displayed.
elements.lng false Tag within which the longitude will be displayed.
elements.locality false Tag within which the city will be displayed.
elements.administrative_area_level_1 false Tag within which the administrative area level 1 will be displayed.
For example States in USA, "Région" in France.
elements.administrative_area_level_2 false Tag within which the administrative area level 2 will be displayed.
Departments in France.
elements.country false Tag within which the country will be displayed.
elements.zip_code false Tag within which the zip code will be displayed.
elements.type false Tag within which the view type will be displayed. For example: locality, establishment, street address, etc.
callbacks Object Callbacks list.
callbacks.before_load false Callback function called before elements (map, etc) loading.
callbacks.after_load false Callback function called at the end of elements loading.
callbacks.before_update false Callback function called before doing informations's (latitude, longitude, zip code, etc) update .
callbacks.after_update false Callback function called once informations's (latitude, longitude, code postal, etc) update have been done.
callbacks.before_move false Callback function called before the marker's move.
callbacks.after_move false Callback function called after the marker's move.
callbacks.on_focus_address false Callback function called when the address suggested by the autocomplete is focused.
callbacks.on_select_address false Callback function called when the address suggested by the autocomplete is selected.

Documentation (Français)

Nom de l'option Valeur par Défaut Description
address_suffix "" (chaine de caractères vide) Suffixe à rajouter aux adresses lors de l'envoi au géocodeur.
Exemple: ", France" si vous ne voulez que des adresses française.
bind_marker true Afficher un marqueur sur la map.
draggable_marker true Donner la possibilité à l'utilisateur de déplacer le marqueur sur la map.
region_bias null Influencer le résultat renvoyé par le geocodeur en fonction d'une région.
Par exemple, si region_bias est à "fr", une recherche sur Paris renverra la capitale française et non son homonyme du Texas ou du Tennessee.
map_options Object Options concernant la map et son initialisation.
map_options.min_zoom 4 Niveau de zoom minimum
map_options.max_zoom 18 Niveau de zoom maximum.
map_options.tile.url "http://{s}.tile.cloudmade.com/BC9A493B41014CAABB98F0471D759707/997/256/{z}/{x}/{y}.png"
map_options.tile.attrib "Map data © openstreetmap contributors"
map_options.zoom 5 Niveau de zoom initial.
map_options.center new L.LatLng(46, 2) Centrage par défaut.
map_options.scroll_wheel false Autoriser le zoom sur la map au scroll.
elements Object Liste des Balises contenant les données à mettre à jours.
elements.map false Balise accueillant la map.
elements.lat false Input dans lequel sera affichée la latitude.
elements.lng false Input dans lequel sera affichée la longitude.
elements.locality false Input dans lequel sera affichée la ville.
elements.administrative_area_level_1 false Input dans lequel sera affichée la zone administrative de niveau 1.
État aux USA, Région en France.
elements.administrative_area_level_2 false Input dans lequel sera affichée la zone administrative de niveau 2.
Département en France.
elements.country false Input dans lequel sera affiché le pays.
elements.zip_code false Input dans lequel sera affiché le code postal.
elements.type false Input dans lequel sera affiché le type de la vue. Par exemple: localité, établissement, adresse, etc.
callbacks Object Liste des callbacks.
callbacks.before_load false Fonction de callback appelée avant le chargement des éléments (notamment de la map).
callbacks.after_load false Fonction de callback appelée une fois le chargement des éléments (notamment de la map) effectué.
callbacks.before_update false Fonction de callback appelée avant la mise à jour des informations (latitude, longitude, code postal, etc).
callbacks.after_update false Fonction de callback appelée une fois la mise à jour des informations (latitude, longitude, code postal, etc) effectuée.
callbacks.before_move false Fonction de callback appelée avant le déplacement du marker.
callbacks.after_move false Fonction de callback appelée après le déplacement du marker.
callbacks.on_focus_address false Fonction de callback appelée au focus sur l'adresse proposée par l'autocomplete.
callbacks.on_select_address false Fonction de callback appelée à la sélection de l'adresse.

About the project

Le projet jQuery Leaflet addresspicker est inspiré de jQuery addresspicker qui fait globalement la même chose, mais pour google maps.

Je travaillais sur un projet leaflet et voulais avoir quelque chose comme jQuery addresspicker, mais en légèrement plus complet, et adapté à leaflet. C'est ainsi qu'est né le projet jQuery Leaflet addresspicker.

Si vous voulez contribuer, faites un fork du projet et envoyez une pull request :).

The project jQuery Leaflet addresspicker was inspired by jQuery addresspicker which is nearly the same plugin for google maps.

I was working on leaflet and wanted to have something like jQuery addresspicker in a little bit more efficient, so i did leaflet addresspicker.

If you want to contribute, please fork the project and do a pull request :).
